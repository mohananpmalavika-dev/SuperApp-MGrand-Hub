## 🚀 Quick Start Guide

### **1. Installation**
```bash
cd frontend
npm install
```

### **2. Configuration**
```bash
cp .env.example .env
```

Edit `.env`:
```env
REACT_APP_API_URL=http://localhost:4000
REACT_APP_RAZORPAY_KEY_ID=rzp_test_your_key_here
```

### **3. Start Development Server**
```bash
npm start
```

### **4. Build for Production**
```bash
npm run build
```
